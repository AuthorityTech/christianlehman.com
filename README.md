# christianlehman.com

Personal site and Invisible Shortlist blog — Christian Lehman, Co-Founder of AuthorityTech.

Blog share images are generated during `prebuild` by `scripts/generate-og-images.mjs`. The canonical image for every post is `public/images/{slug}.png`; pages, metadata, and schema derive that URL from `src/lib/postShare.ts` instead of per-post frontmatter.
